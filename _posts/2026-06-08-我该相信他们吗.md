## 技术上

### itedev.com

熟悉我的 Bro 都知道, 我使用的是 Cloudflare Business 订阅, 而 Business 订阅赋予了我一项权利: **CNAME 接入**, 就像这样.

![DNS 设置:部分](https://bucket.itedev.com/blog/2026/05/25/87bc8bb890375a86a9bcd105655cf5360a10b374.1779702605.png)

而我的 NS, 用的是~~遥遥领先~~的华为云. 有人就会说: "主播主播, 你 NS 接入用的好好的, 为什么要花大力气切到 CNAME 呢?", 问得好, 可以看看下面的表格, 罗列的我的权衡(并非拉踩比较).

|          |                           NS 接入                            |                          CNAME 接入                          |
| :------: | :----------------------------------------------------------: | :----------------------------------------------------------: |
|   DNS    |    可以享受 Cloudflare 提供的许多记录类型, 免费的 DNSSec     | 华为云的记录类型没 Cloudflare 全, 且在 Google Trust Service 发证书时会出现地区 CAA 错误 |
|  可靠性  | 你见过 Cloudflare 的 CDN 炸, 见过它的 DNS 解析炸吗(我是没见过的暂时)? |       华为云不清楚, 但是没有在社区见过大规模停机的事情       |
|   费用   |                             免费                             |                     Business 及以上免费                      |
|   支持   | Free 靠社区和 doc, Pro 靠工单, Business 及以上可以与活人直接对话 | 华为态度超级超级好, 甚至开完工单会直接打电话来问你细节, 见[关于 CNAME 接入服务的故障报告](https://www.itedev.com/posts/2025/12/21/%E5%85%B3%E4%BA%8E-CNAME-%E6%8E%A5%E5%85%A5%E5%9F%9F%E5%90%8D%E7%9A%84%E6%95%85%E9%9A%9C%E6%8A%A5%E5%91%8A) |
|   文档   | 详实, 有主动的 Agent 支持(但是我运维靠手动, 不会使用 Agent)  | 至少我选的华为云, 文档详实, 但是给人看居多, Agent 没有专门的文件 |
| 规范遵守 |    极好, 但是注入记录在仪表板中不显示(如 CAA, SOA一类的)     |                     华为云我目前认为极好                     |
|   CDN    |       只有代理和非代理二元状态, 在做自选的时候及其不便       |                   就正八经的传统 CDN 体验                    |
|   灾备   | 假设 Cloudflare 的 DNS 死了, 解析不动; 假设 CDN 死了, 你想把流量导向别家, 会很麻烦 |                  传统 CDN 体验, 关记录就行                   |
| 多提供商 |        区域记录同步只对 Enterprise 启用(对, 我用不到)        |         区域记录同步对所有人启用(免费, 如果我没记错)         |

现在来解释一下为什么 NS 接入在实现类似于优选以及更换提供商时会比较麻烦: 

1. Cloudflare 在 Free 是不允许出现别家 CDN 的记录的(至于是代理时还是关闭代理时, 我忘了)
2. Cloudflare 想实现优选, 需要确保记录为非代理, 且你需要配置 Worker 路由或者 Saas 接入来确保流量被 Cloudflare 边缘捕获
3. 如果想要只用一个域名解决优选, 会出现 NS 委派, DNSSEC 配置等麻烦, 以及非常雷霆的 CNAME 链, 就像这样:`用户 -(1)-> www.example.com -(2)-> www.example.com.saas.example.com --> fallback记录或者其他 CNAME 记录`. 这会产生非常多的递归查询, 效率极低(包含两次对 Cloudflare NS 的查询, 大陆灾难), 且无法确保境内的运营商 DNS 会不会出现未定义行为(我默认所有域都启用 DNSSEC)
4. 在 Cloudflare 用非代理记录 CNAME 出去后, 需要进行 DCV 委派以保证 Cloudflare 颁发证书, 麻烦加一
5. 如果 Cloudflare 号炸了, CNAME 方式会更好迁移
6. 如果你没看懂上面的内容, 这也是一个麻烦, 因为运维难度会很高, 容易写错记录/做错步骤导致停机, 我的设施承载着一定的 Saas 流量, 我不想保留风险.

总而言之, 为了自己的工作轻松, 为了**我们都拥有美好的未来**, 我选择了 CNAME 接入.

## 生活上

熟悉我的读者都知道, 我的前 12 年, 过得并不算好. 至于发生什么事情, 在这里再次提及, 我认为是不恰当且非常煞风景的. 那为什么我会在这一段以这一句开头呢? 我在学校与同学的关系并不算好, 也许因为我是个精神病(?), 又或者因为我太牛逼了, 又或者因为我**唐**, 还是单纯看我不顺眼? 这些我都不在意, 但是我在意的是, 自己不见了东西赖我, 自己搞脏了我的床, 反而给我扣帽子说我偷吃东西……

我一直对你们是容忍态度, 我也不想就此事与你们辩论, 身正不怕影子斜. 但是因为这件事情, 我也曾经低落过一段时间, 并提出了转校的想法, 故事也从这里开始.

我从我在某所学校的一位初中同学, 了解到, 他们班的班风, 乃至整个年级的风气, 都不似我所在的这所学校这样(我在区一中, 这位同学在市一中, 由于是市级的顶尖学府, so生源素质理应高点). 于是, 我萌生了转学到这所学校的想法, 并和家里谈判了许久, 得到了**肯定**的回答, 并表示会尽力帮助我.

在一天早上, 我起床了, 但是我还我在被窝里睡懒觉(这是个周末). 由于我恰好醒的比较早, 便听到家长的谈话. 谈话内容我不愿再次复述, 但是大意就是怎么伪造已经打了电话问过招办的记录, 来让我死心. 我感觉不妙, 但是他们似乎谈完了, 又或者察觉到有人在听, 遂停止谈论, 出去买菜.

又隔了几天, 一个工作日, 他们很主动的来告诉我, 没戏了, 并展示了通话记录, 还让我听了通话录音. 但是记录是怎么样的呢? 通话时间仅 10s, 录音则是他们工作场所的白噪音. 但是当时我不好说什么, 便笑脸相迎, 和他们讲我了解了, 打发他们离开我的房间. 但是那天晚上, 我彻夜难眠, 第二天早上, 我笑了笑, 又去上学.

这一笑, 像是在笑我自己又一次相信了他们; 也在笑自己的颓唐, 真的认为自己那么厉害能改变上层建筑呢(因为经济基础决定上层建筑, so相对于我, 家长更像我上面的一个"国家机器", 仅会意, 并非严谨意)……

### 女朋友, 女朋友?

我在很早之前立过一个 flag, 写一篇《**青少年恋爱指南**》, 你可以[在这里](/posts/2026/01/02/年终总结)找到原话. 上图的那位女生, 在澳门内部某间喜茶附近等我们的队友去买喜茶的时候, 曾向我透露的这位可怜的队友的情感史. 我也顺水推舟, 套出了她的一部分情感史(如果那位可怜的队友看到了这篇博客, 请不要看见这篇博客, 我相信你不会那么闲来光顾我的网站的╰(*°▽°*)╯).

我在每见到一对情侣的时候, 如果他们或他们之中的一个人和我聊天, 我必定会问两个问题:

1. 你们有没有想过以后怎么样, 比如异地/结婚/分手一类的;
2. 你们觉得你们能持续多久.

这位与我们同行的女生, 在我说完上面那一段之后, 非常直接的点出了: "就算我们有思考, 那我们也不会告诉你啊". 说的没错, 确实不会告诉我, 这是你们之间的很私密的事情. 但我还是要问, 或者是提一嘴, 我认为这两个问题有利于关系的长久健康稳定发展, 或者, 换个说法, 总会遇到的——早点想好这个问题反而益处更多.

我认为, 作为一个男性, 如果想要找到属于自己的另一半, 那就必须要从一个**男孩**, 转变成为一个**男人**. 成为男人, 即需要承担属于自己的那一份责任, 相信想看我写出恋爱指南的青少年, 想早恋的青少年, 或多或少都有这种觉悟.

我对早恋持保留态度, 甚至也甜甜的希望有另一半陪着我. 但是我看过一个抨击早恋的视频, 里面说的内容我却非常同意

> 青少年(18以下)的恋情, 很大程度上是看脸的恋情

诚然, 在未成年时, 没有经济能力, 你不知道以后自己能不能撑起一个家. 拜金, 同时也是部分女性的择偶标准. 这些事情, 在未成年时, 或未自己劳动所得赚过钱, 不知道赚钱困难的, 即使彪炳自己再成熟, 也是无济于事的. 对劳动的敬畏, 对劳动者的同情与共情……

我想在此引出一个叫做小市民的概念, 在以前, 我对其的印象是"粗俗""市侩""油腻圆滑"一类的贬义; 但随着自己越长越大, 开始用自己赚的钱帮家里还房贷, 度过生活难关时, 我也曾不止一次的发现, 我又何尝不是那个小市民呢? 自那以后, 我对这个词便改观了. 在小时候, 总想着自己独立, 总想着自己远走高飞. 长大以后, 才理解家里的难处, 家长的难处. 既然这个家没有雷霆到不可住人的地步, 我又为啥要走呢? 更何况, 我在用我的力量一步一步的将它变好.

有人可能会说, 你就不怕家人再阴我一手吗? 我怕啊, 我一直都怕啊. 所以我有自己独立开的情绪处理机制, 我可以在极大的压力下仍保持理智, 很多过去的我的事实, 实际上证明这一点. 我卡里有钱, 我给家里钱, 也不会完全透支我自己的生活资本, 我养活自己了, 再养家, 这是我的观点.

老一辈家长之所以十分令人讨厌, 是因为他们雷霆到发指的控制欲, 是因为三纲五常, 是因为…… 但是, 可恨之人必有可怜之处, 他们何尝不是走这条路过来的呢. 当我们成为他们, 我们会对自己的孩子怎么样. 亦或者, 有弟弟妹妹侄子侄女的, 在带小孩的时候, 也或多或少感受过这些磨砺.

只有把这些磨砺都经历过一次之后, 我觉得一个男人才算真正炼成. 男人生子大多近 30, 孩子长到青春期大概 40 几, 故有:

> 三十而立, 四十不惑

只要你看起来像个男人了, 你的性格特质看起来像个男人了, 不需要你生过小孩或者什么, 只要有体谅自己上一辈(如果他们没有坏到极致)的能力, 共情或者换位思考的能力, 保持边界(这点很重要), 自然会有女生追(当然我不是权威军师不包好使).

以上是比较现实的打法, 我认识的人中, 也有意识流的打法, 但是我认识的那个意识流女生(她的原话: 我是个偏主观的人, 意思是她脑里面有自己的世界), 并没有成家的打算, 也没有任何谈恋爱的打算, 故我没有样本, 不够格写这段.

不过我认为, 从男孩到男人的过程, 从女孩到女人的过程, 无论是现实流还是意识流, 都会收敛到一个点, 一个现实为导向的点. 至于那些理想主义者, 他们不也是在现实的指导下才产生对现实社会的理想吗.

需要异性或者同性或者其他性别的另一半, 内在需求, 要么是性, 要么是需要有一个港湾可以卸下一切防备, 拜金主义一类的暂不在讨论范畴之内. 所以只要一是解决性需求, 二是解决自己的心里需求, 就可以抛掉另一半(这点仅给希望"佛系"或者"待定, 等对的人"的人用, 并没有宣扬什么奇奇怪怪的内容).

写到这里已经力竭了, 剩下的就靠自己会意吧.
